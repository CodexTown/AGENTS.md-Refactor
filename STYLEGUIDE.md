# Python Style Guide for Project Omega

1. **Type Hinting**: All functions must have type hints for arguments and return values.
2. **Docstrings**: All functions must have a docstring explaining what they do, args, and returns.
3. **Naming**: Use descriptive variable names (no single letters like `x`, `d`, `l`).
4. **Structure**: Separate logic into small, single-purpose functions.
5. **Entry Point**: Use `if __name__ == "__main__":` block.
